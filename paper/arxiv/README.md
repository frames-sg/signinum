# signinum arXiv source package

This directory contains the arXiv-oriented preprint source for `signinum`.
It is intentionally separate from `paper/paper.md`, which remains the JOSS
manuscript.

## Metadata

- Title: `signinum: WSI-shaped JPEG and JPEG 2000 codecs for digital pathology`
- Primary category: `cs.SE`
- Suggested cross-listing: `eess.IV` if arXiv permits it for the submission
- License: Creative Commons Attribution 4.0 International (CC BY 4.0)
- Repository: <https://github.com/jcwal1516/signinum>
- Version discussed: `0.1.0`

## Build

Use TeX Live with `latexmk`, matching arXiv more closely than Pandoc-only
conversion:

```sh
cd paper/arxiv
latexmk -pdf -interaction=nonstopmode -halt-on-error main.tex
```

If `latexmk` is missing on macOS, install BasicTeX or MacTeX and make sure the
TeX binaries are on `PATH`.

## Verification checklist

Before upload:

```sh
cargo xtask typos
cargo xtask test
cargo xtask bench-build
git diff --check
```

Then inspect `main.pdf` manually:

- Title, author, abstract, figures, and tables fit within margins.
- Citations and references render correctly.
- The two private-slide timing examples are clearly labeled as local
  whole-reader integration evidence, not public reproducible codec benchmarks.
- CUDA wording states fallback-only compatibility for version `0.1.0`.
- The source package contains `main.tex` and `references.bib`; if a local TeX
  build also produces `main.bbl`, it may be included because arXiv treats
  processed bibliography files as an allowed exception.

## arXiv upload contents

Upload only source files needed to build the manuscript:

- `main.tex`
- `references.bib`
- optional `main.bbl`, if generated by the same BibTeX build used for the PDF

Do not upload generated files such as `*.aux`, `*.blg`, `*.log`, `*.out`,
`*.fls`, `*.fdb_latexmk`, `*.synctex.gz`, or `build/`.
